# 🌿 Plant Disease Detection using CNN & Gemini API

## 📌 Introduction
Agriculture plays a vital role in food security, but plant diseases cause major crop losses. Traditional detection methods are slow and require expert knowledge.  
This project uses **Artificial Intelligence** to detect plant diseases quickly and accurately by combining a **CNN model** with **Google Gemini API** for real-time insights.

---

## 🎯 Problem Statement
- Farmers face income loss due to late disease detection  
- Manual methods are slow, costly, and not practical for large farms  
- Need for an **automated, accurate, and real-time detection system**

---

## 🎯 Objectives
- Develop a CNN model for plant disease classification  
- Integrate Gemini API for disease details and treatment  
- Build a Streamlit web app for easy usage  
- Provide multilingual output for farmers  

---

## 🚀 System Overview
- 📂 **CNN Model** → Disease prediction  
- 🤖 **Gemini API** → Description & treatment  
- 🌐 **Streamlit App** → User interface  

---

## 🛠️ Technology Stack
- Python

- Google Gemini API

## 📊 Dataset
